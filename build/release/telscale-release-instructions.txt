Doing a Release : 

Release the components below if there was modifications since the previous release

For all releases below make sure to do :
git checkout ts1
git clean -df
git status (make sure there is nothing to commit and clean)

and After each release, don't forget to do git checkout ts1ga and mvn clean install as the release:prepare don't build the project

Release JSIP first :
cd m2/jain-sip-ri
mvn release:prepare --batch-mode -DdevelopmentVersion=6.1.1-TelScale-SNAPSHOT -DreleaseVersion=6.1.0.GA-TelScale -Dtag=ts1ga
